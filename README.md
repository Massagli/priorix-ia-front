# 🎂 Priorix IA

Sistema inteligente para gerenciamento de encomendas de confeitaria.

O projeto possui:

- Dashboard inteligente
- Sistema de prioridades
- Chat IA para cadastro de pedidos
- Cronograma automático
- Integração completa com FastAPI

# 🚀 Tecnologias

## Frontend
- HTML
- CSS
- JavaScript

# ▶️ COMO RODAR O PROJETO

O sistema possui:

✅ Backend FastAPI  
✅ Frontend HTML/CSS/JS  

Os dois precisam estar rodando ao mesmo tempo.


# 1️⃣ Rodar o Backend (FastAPI)

Abra um terminal dentro da pasta do backend:

```bash
cd backend
```

---

## Instale as dependências

```bash
pip install fastapi uvicorn sqlalchemy pydantic
```

---

## Inicie a API

```bash
uvicorn main:app --reload
```

---


# 2️⃣ Rodar o Frontend

Abra OUTRO terminal na pasta do frontend:

```bash
python -m http.server 8080
```

---

## O frontend ficará disponível em:

```txt
http://127.0.0.1:8080
```

ou

```txt
http://localhost:8080
```
